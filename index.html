História do HTML

O HTML (Linguagem de Marcação de Hipertexto) foi criado por Tim Berners-Lee nos anos 90 com o objetivo de facilitar a criação de páginas web. Com o tempo, evoluiu para suportar diversos tipos de conteúdo e se tornou essencial na internet.

O que são Tags?

Tags são comandos em HTML usados para marcar e estruturar o conteúdo de uma página. Por exemplo, usamos a tag <p> para parágrafos e <h1> a <h6> para títulos.

Tags funcionam com abertura e fechamento: <b>Texto</b>.

Estrutura básica do HTML

Todo documento HTML começa com <!DOCTYPE html> e contém as tags principais como:

  • <html>: raiz do documento
  • <head>: metadados
  • <body>: conteúdo visível

Listas ordenadas e não ordenadas

Listas ordenadas usam a tag <ol> e criam uma sequência numerada:

  1. Primeiro item
  2. Segundo item

Listas não ordenadas usam a tag <ul> e exibem marcadores:

  • Item A
  • Item B
